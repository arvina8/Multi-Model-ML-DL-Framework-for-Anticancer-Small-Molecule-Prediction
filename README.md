
# Multi-Model ML–DL Framework for Anticancer Small-Molecule Prediction

## Overview
This project presents a hybrid machine learning and deep learning framework for predicting anticancer activity of small molecules. The system integrates multiple models to improve prediction accuracy and generalization.

## Features
- Multi-model architecture (ML + DL)
- Feature engineering for molecular data
- Model comparison and evaluation
- Scalable prediction pipeline

## Models Used
- Random Forest
- XGBoost
- Deep Neural Networks (DNN)

## Technologies
- Python
- Scikit-learn
- TensorFlow / PyTorch (if used)

## Results
- Improved predictive performance through model combination
- Comparative analysis across models

## How to Run
1. Clone the repository
2. Install dependencies
3. Run the main script / notebook
